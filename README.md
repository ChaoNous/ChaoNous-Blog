# ChaoNous Blog

> 下一代 Material Design 3 博客主题，由 Astro 驱动

## 特性

- **Material Design 3** - 现代化设计语言，支持深色/浅色模式
- **Astro + Svelte** - 高性能静态站点，按需水合
- **Tailwind CSS** - 原子化 CSS，灵活定制
- **全文搜索** - Pagefind 驱动的本地搜索
- **代码高亮** - Expressive Code 支持，支持折叠、行号
- **RSS/Atom/Sitemap** - 完善的 SEO 支持
- **相册/书籍/时间线** - 丰富的内容展示
- **Swup 页面切换** - 流畅的页面过渡动画

## 本地开发

```bash
pnpm install
pnpm dev
```

## 构建

```bash
pnpm run check
pnpm run build
```

## 部署

推送到 `master` 分支，Cloudflare Pages 自动构建部署。
